Archives & Catalog Retrieval

The Archives & Catalog Retrieval dialog allows access to remote image and star catalod services.

Use the menus in the top left to select a data source.

Pressing Retrieve Data in the upper right will start the query.

If the Object field is not blank the retrieval will try to convert the object name to an RA and Dec before placing a query.

If an image is already loaded in to the JS9 frame, the Set RA/Dec button will retrieve the field center and size from the image and fill in the dailog's form fields.
